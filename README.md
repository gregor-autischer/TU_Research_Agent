# TU Research Agent

A Django + Vue.js research assistant application.

## Quick Start

```bash
docker compose up --build
```

The app will be available at http://localhost:8009

## Default Credentials

- Username: `admin`
- Password: `admin`

The admin user is created automatically on first startup.

## Admin Panel

Access the Django admin at http://localhost:8009/admin to view and manage the database.
